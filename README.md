# 🚀 AI Surveillance System

An AI-powered surveillance system that detects weapons in real-time using computer vision and a web dashboard.

---

## 🔥 Features

- 📹 Live camera streaming (IP Webcam)
- 🧠 AI-based object detection (YOLO)
- 🚨 Real-time weapon detection alerts
- 🌐 Web dashboard (Flask)
- 📊 Multi-camera support

---

## 🛠️ Tech Stack

- Python
- Flask
- OpenCV
- YOLO (Ultralytics)

---

## 🚀 How to Run

```bash
git clone https://github.com/your-username/AI-Surveillance.git
cd AI-Surveillance

python -m venv ai_project
source ai_project/bin/activate

pip install -r requirements.txt

cd src
python app.py
